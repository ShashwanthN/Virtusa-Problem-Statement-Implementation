# job-service

Spring Boot service that manages `Job` lifecycle and orchestrates external quiz + python AI services.

## Endpoints

- `POST /api/jobs`: creates a job and saves quiz via `quiz-service` (`autoGenerateQuiz=true` means quiz is generated by python backend first, then saved in quiz-service).
- `GET /api/jobs`: returns all jobs.
- `POST /api/jobs/apply` (`multipart/form-data`): forwards resume to `python-ai-service`, receives matched job IDs, and returns matched jobs.
- `POST /api/jobs/{jobId}/start-test`: returns test time and questions.
- `POST /api/jobs/{jobId}/submit-test`: validates answers and returns summary + applied status.

## Config

See `src/main/resources/application.properties` for environment-overridable properties:

- DB: `JOB_DB_URL`, `JOB_DB_USERNAME`, `JOB_DB_PASSWORD`
- Quiz: `QUIZ_SERVICE_BASE_URL`, `QUIZ_SERVICE_CREATE_PATH`, `QUIZ_SERVICE_START_TEST_PATH`, `QUIZ_SERVICE_SUBMIT_TEST_PATH`
- Python AI: `PYTHON_AI_SERVICE_BASE_URL`, `PYTHON_AI_MATCH_PATH`, `PYTHON_AI_INDEX_PATH`, `PYTHON_AI_GENERATE_QUIZ_PATH`

## Run

```bash
./mvnw spring-boot:run
```
